## 更新/套件安裝



>1. 更新套件清單
>
>2. 更新套件 
>
>3. 安裝apt套件 (示例: cowsay 文圖顯示公告資訊)


```
1. sudo apt update

2. sudo apt upgrade

3. sudo apt install cowsay

格式: cowsay -f [動物名ex: tux] “輸入要動物說的話” 
Example: cawsay -f tux “Hello”| lolcat

Tip: |符號是窜接指令的意思，lolcat表示秀出彩色。

```
